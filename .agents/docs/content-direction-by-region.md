# 지역별 콘텐츠 방향

Pick Trip은 유명 관광지보다 사용자가 직접 선택하고 일정으로 연결할 수 있는 지역 콘텐츠를 우선한다.

## 공통 원칙

- 장소의 유명도보다 일정 구성에 실제로 쓸 수 있는 정보를 우선한다.
- 대표 이미지, 소요 시간, 운영 시간, 위치, 계절성, 동행 적합성을 노출한다.
- 콘텐츠 설명은 과장보다 방문 판단에 필요한 구체 정보를 제공한다.
- 지역별 카테고리 균형을 유지해 음식, 자연, 문화, 축제가 한쪽으로 치우치지 않게 한다.

## 하동

| 방향 | 예시 |
| --- | --- |
| 자연 | 섬진강, 녹차밭, 지리산 자락 |
| 음식 | 재첩, 참게, 녹차 기반 디저트 |
| 문화 | 전통 마을, 문학 관련 장소 |
| 축제 | 봄꽃, 차 문화, 지역 농산물 행사 |

하동은 자연 경관과 느린 여행 흐름을 강조한다. 가족, 커플, 중장년 여행자가 이동 피로를 줄이며 둘러볼 수 있는 동선을 우선한다.

## 영주

| 방향 | 예시 |
| --- | --- |
| 문화 | 부석사, 선비촌, 소수서원 |
| 자연 | 소백산, 계곡, 숲길 |
| 음식 | 한우, 사과, 지역 농산물 |
| 체험 | 전통문화, 역사 교육형 콘텐츠 |

영주는 역사와 자연을 함께 묶는 일정에 강점이 있다. 문화유산 설명은 짧고 명확하게 제공하고, 학습형 여행과 가족 여행에 맞는 정보를 강화한다.

## 예천

| 방향 | 예시 |
| --- | --- |
| 자연 | 회룡포, 강변 경관, 생태 명소 |
| 문화 | 전통 마을, 지역 역사 자원 |
| 음식 | 지역 농산물과 향토 음식 |
| 체험 | 활, 곤충, 생태 체험형 콘텐츠 |

예천은 생태와 체험 중심으로 구성한다. 아이 동반 가족 여행자가 이해하기 쉬운 콘텐츠와 짧은 이동 동선을 우선한다.

## UI 표현 기준

- 지역별 색상이나 이미지 톤은 보조 요소로만 사용하고, 정보 탐색성을 해치지 않는다.
- 카드는 지역, 카테고리, 소요 시간, 선택 상태를 빠르게 비교할 수 있어야 한다.
- 상세 화면은 일정에 담기 전 판단에 필요한 핵심 정보를 상단에 배치한다.
